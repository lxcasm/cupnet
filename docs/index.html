Fonctionnalités

ARP + ping : IP, MAC, fabricant, latence et temps sur le réseau.

Coupe bidirectionnelle cible ↔ box. Nécessite admin + Npcap.

Surveillance

Temps de blocage en direct, détail par appareil, session active.

Confidentialité

Option pour masquer les IP dans le tableau et le réseau affiché.

Installation

  1. Téléchargez le projet depuis GitHub (bouton Code → ZIP).
  2. Installez Npcap et Python 3.10+ si ce n'est pas déjà fait.
  3. Double-cliquez sur CupNet.bat à la racine du dossier.
  4. Si Npcap n'est pas installé, CupNet propose de le télécharger et l'installer au lancement.
  5. Acceptez UAC (admin) pour activer la coupure réseau ARP et l'installation Npcap.
  6. Cliquez Scanner le réseau, sélectionnez un appareil, coupez ou débloquez.

Aucun .exe à compiler : CupNet.bat installe et lance tout automatiquement.

Options ⚙

Dans l'app, cliquez sur l'engrenage en haut à droite :

  • Masquer les IP — masque les adresses dans le tableau et le réseau en haut
  • Agrandir le tableau — plus d'appareils visibles à l'écran
  • Temps de chargement — splash au démarrage (0 à 5 secondes)
  • Installer Npcap — bouton visible uniquement si Npcap est absent
  • Npcap — proposé au démarrage si absent (optionnel, requis pour scan/ARP)
